### Download

The GNU/Linux versions of **xPack Ninja Build**
are packed as `.tar.gz` archives.
Download the latest version named like:

* <code>xpack-ninja-build-{ customField('upstreamVersion') }-{ customField('xpackSubversion') }-linux-x64.tar.gz</code>
* <code>xpack-ninja-build-{ customField('upstreamVersion') }-{ customField('xpackSubversion') }-linux-arm.tar.gz</code>
* <code>xpack-ninja-build-{ customField('upstreamVersion') }-{ customField('xpackSubversion') }-linux-arm64.tar.gz</code>

:::note

In case you wonder where the suffix comes
from, it is exactly the Node.js `process.platform` and `process.arch`.

:::

### Unpack

To manually install the <b>xPack Ninja Build</b>,
unpack the archive and move it
to <code>~/.local/xPacks/ninja-build</code> folder, for example
<code>/home/ilg/.local/xPacks/ninja-build/xpack-ninja-build-{ customField('upstreamVersion') }-{ customField('xpackSubversion') }</code>:

<CodeBlock language="console"> {
`mkdir -p ~/.local/xPacks/ninja-build
cd ~/.local/xPacks/ninja-build

tar xvf ~/Downloads/xpack-ninja-build-${ customField('upstreamVersion') }-${ customField('xpackSubversion') }-linux-x64.tar.gz
chmod -R -w xpack-ninja-build-${ customField('upstreamVersion') }-${ customField('xpackSubversion') }`
} </CodeBlock>
